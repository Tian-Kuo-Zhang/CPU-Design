# 03 Unit Test - RF and RAM

## 目标

验证寄存器堆和数据存储器的基本契约：

- RF: 异步读、同步写、`R0` 恒为 0
- RAM: 同步写、异步读

## 操作步骤

### RF

1. 写 `R1=16'h1234`，下一拍读取 `R1`。
2. 尝试写 `R0=16'hFFFF`，读取 `R0`。
3. 同时读两个端口（例如 `R1` 和 `R0`）。

### RAM

1. 地址 `0x0010` 写入 `0xABCD`。
2. `MemRead=1` 读同地址，检查返回。
3. `MemRead=0` 时检查输出是否回到 `16'h0000`（按当前实现）。

## 预期结果

- RF: `R1` 能被写入，`R0` 永远读为 0。
- RAM: 写后可读，且读地址只取低 8 位。

## 能抓到的 bug

- RF 对 `R0` 保护缺失。
- 写使能时序错误（本拍写本拍读出现不一致）。
- RAM 地址截断导致高位地址别名冲突（`addr[15:8]` 被忽略）。
